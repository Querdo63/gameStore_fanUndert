# Гайд по запуску

Можете создать виртуальное окружение, чтобы подгрузить все зависимости отдельно от основной системы (а можете не создавать, но в будующем могут быть неприятности):

```cmd
python -m venv venv
```

---

Разумеется если создали окружение, то нужно в него войти:

```cmd
venv\Scripts\activate
```

---

Подгружаем зависимости (библиотеки):

```cmd
pip install -r requirements.txt
```

Запускаем бэкенд:
```cmd
uvicorn main:app --reload
```

Открываем отдельную консоль и запускаем фронтенд (да, это запуск локального сервера, потому что просто открыть файл index.html может быть недостаточно):
```cmd
python -m http.server 3000
```
